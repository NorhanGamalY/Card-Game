# Card Game

This is a web-based card game developed using TypeScript and HTML. The game features customizable decks, interactive gameplay mechanics, and a responsive user interface.

## Features

- **Customizable Deck**: Players can create and manage their own deck of cards.
- **Interactive Gameplay**: Play the card game with smooth animations and interactions.
- **Responsive UI**: The application is designed to work seamlessly across devices.
- **Game Logic**: Reusable modules for managing game state, turns, and player actions.
- **Asset Integration**: Includes audio and image assets for a more immersive experience.
